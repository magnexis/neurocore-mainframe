# UI System

This document describes the visual system for a private prototype. It is not a public design system, contribution guide, or release specification.

The interface is designed to feel like a secure system surface rather than a conventional marketing site.

## Palette

- Background: `#000000`
- Primary signal: `#FF2D55`
- Green accent: `#00FF88`
- Purple accent: `#A070FF`
- Text: `#ffe4ea`

## Typography

- Interface text uses Inter.
- System labels, logs, commands, and data use IBM Plex Mono.

## Components

- Panels use a thin neon border, dark translucent fill, and restrained internal glow.
- Navigation buttons use monospace labels and numeric module codes.
- Status pills communicate system health without adding excessive decoration.
- Inputs and command surfaces keep high contrast and clear focus states.

## Responsiveness

Desktop uses three columns: module menu, workspace, and telemetry rail. Tablet removes the rail. Mobile stacks the shell and turns module navigation into a compact grid.
