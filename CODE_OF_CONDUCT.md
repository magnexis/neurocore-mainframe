# Code of Conduct

NeuroCore // MAINFRAME is a private creative engineering prototype. It is not open for public contributions, issue discussion, or community maintenance.

This code of conduct exists only as a basic expectation for any private review or local handoff context. It should not be interpreted as an invitation to open discussions, submit pull requests, or participate in a public project community.

## Expected Behavior

- Be considerate with feedback.
- Assume good intent when clarifying requirements or reviewing code.
- Keep criticism specific to code, design, documentation, or project decisions.
- Respect privacy and avoid sharing personal or sensitive information.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private information without consent.
- Disruptive comments unrelated to the project.
- Attempts to use the project to misrepresent real surveillance, email access, or security capabilities.

## Enforcement

Maintainers may remove content, close discussions, or restrict participation when behavior conflicts with this code of conduct.
