# Contributing

NeuroCore // MAINFRAME is not open for outside contribution.

This repository is maintained as a private creative prototype and local demonstration artifact. Pull requests, feature requests, issue reports, roadmap proposals, release requests, and community maintenance offers are not expected or accepted.

The documentation is included to explain how the project is structured and how it can be run locally. It should not be read as an invitation to contribute.

If you fork or adapt the project under the terms of the license, maintain your changes independently.
